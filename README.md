# Демо https://hecathe.github.io/mebel-corpus/

## Использование

Устанавливаем

```
npm install
```

Сборка для разработки

```
gulp
```

Сборка для продакшена

```
gulp prod
```

Обновить библиотеки

```
gulp up-lb
```
